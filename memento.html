Memento-suunnittelumalli Pythonissa

Mikä on Memento?

Memento-suunnittelumalli mahdollistaa olion sisäisen tilan tallentamisen ja palauttamisen myöhemmin ilman, että tila paljastetaan ulkopuolisille. Tämä on hyödyllistä mm. kumoa-toiminnallisuudessa, historiassa ja tilanhallinnassa.

Tarkoitus:
Tallentaa ja palauttaa olion tila siten, että kapselointi säilyy.

Mallin osat

  • Originator: Olio, jonka tila halutaan tallentaa ja palauttaa
  • Memento: Tallentaa Originatorin tilan
  • Caretaker: Hallitsee mementoja (esim. tallentaa historiat)

Python-esimerkki

1. Memento-luokka: tallentaa tilan

class Memento:
    def __init__(self, state):
        self._state = state

    def get_state(self):
        return self._state

2. Originator-luokka: tila ja tilan tallennus/palautus

class Originator:
    def __init__(self):
        self._state = None

    def set_state(self, state):
        print(f"Tila asetettu: {state}")
        self._state = state

    def save_state(self):
        print("Tila tallennettu.")
        return Memento(self._state)

    def restore_state(self, memento):
        self._state = memento.get_state()
        print(f"Tila palautettu: {self._state}")

3. Caretaker-luokka: hallitsee tilahistoriaa

class Caretaker:
    def __init__(self):
        self._mementos = []

    def add_memento(self, memento):
        self._mementos.append(memento)

    def get_memento(self, index):
        return self._mementos[index]

4. Käyttö

originator = Originator()
caretaker = Caretaker()

originator.set_state("Tila 1")
caretaker.add_memento(originator.save_state())

originator.set_state("Tila 2")
caretaker.add_memento(originator.save_state())

originator.set_state("Tila 3")

# Palautetaan tila 1
originator.restore_state(caretaker.get_memento(0))

# Palautetaan tila 2
originator.restore_state(caretaker.get_memento(1))

Tulostus:

Tila asetettu: Tila 1
Tila tallennettu.
Tila asetettu: Tila 2
Tila tallennettu.
Tila asetettu: Tila 3
Tila palautettu: Tila 1
Tila palautettu: Tila 2

Hyödyt

  • Mahdollistaa tilan tallentamisen ja palauttamisen kapseloinnin säilyttäen
  • Helppo toteuttaa ja käyttää
  • Tukee toimintoja kuten undo/redo, historian tallennus

Haitat

  • Voi kuluttaa muistia, jos tallennetaan paljon tiloja
  • Suuret tilat voivat hidastaa toimintaa, joten harkinta on tarpeen

Yhteenveto

Osa Kuvaus
Originator Olio, jonka tila tallennetaan
Memento Tallentaa Originatorin tilan
Caretaker Hallitsee mementoja, esim. historiat

Memento-malli on kätevä työkalu, kun halutaan säilyttää olion tila ilman, että paljastetaan sen sisäistä rakennetta. Pythonin joustavuus tekee sen toteuttamisesta sujuvaa ja ymmärrettävää.
